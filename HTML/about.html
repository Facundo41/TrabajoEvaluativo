Acerca de Nosotros

Bienvenidos a Delicias del Mundo, una página dedicada a compartir y descubrir recetas de cocina de todo tipo. Nuestro objetivo es ofrecer una experiencia sencilla, visual y atractiva para que cualquier persona pueda encontrar inspiración culinaria de manera rápida y práctica.

Nuestro Equipo

Somos un grupo de estudiantes. Cada integrante tuvo un rol fundamental en la creación del sitio:

  • Facundo – Desarrollador de la Página de Inicio
    Se encargó de construir la estructura principal del sitio, definiendo la navegación general y el diseño base que unifica todas las secciones.
  • Hugo – Desarrollador de la Página de Contacto
    Diseñó y programó el formulario de contacto, cuidando la accesibilidad, validación de datos y coherencia visual.
  • Eros – Desarrollador de la Página Acerca de Nosotros
    Redactó y diseñó esta sección, enfocándose en reflejar la identidad del grupo y la integración técnica del proyecto.

Temática del Proyecto

La temática elegida es recetas de cocina internacionales, ya que consideramos que la gastronomía es una excelente forma de conectar culturas y fomentar la creatividad. El sitio está pensado para que el usuario pueda buscar, visualizar y aprender recetas de manera dinámica, con un diseño moderno y adaptable.

Decisiones de Diseño

El diseño se basa en una combinación de colores vivos y contrastantes, buscando reflejar la energía y diversidad de la cocina.

  • Se usaron dos paletas de variables CSS para mantener coherencia entre secciones, destacando tonos rojos (#e63946) y azules (#0300ca).
  • La tipografía Inter y Poppins brinda legibilidad y modernidad.
  • Los fondos con imágenes de comidas y los efectos de sombra aportan profundidad visual.
  • El diseño es totalmente responsive, adaptándose a distintos tamaños de pantalla.

API Utilizada

Para obtener la información de las recetas utilizamos la API pública de TheMealDB, específicamente el siguiente endpoint:

https://www.themealdb.com/api/json/v1/1/search.php?s=Arrabiata

Esta API nos permite acceder a una base de datos de comidas del mundo, mostrando imágenes, ingredientes e instrucciones. Su integración fue clave para mantener el contenido actualizado de manera automática.

Detalles Técnicos del Desarrollo

  • Lenguajes utilizados: HTML, CSS y JavaScript.
  • Estructura modular: cada sección fue desarrollada por un integrante y luego integrada en un mismo diseño.
  • Diseño adaptable: mediante el uso de grid, flexbox y media queries.
  • Estilos unificados: usando variables CSS (:root) para colores, márgenes y tipografía.
  • Efectos interactivos: transiciones suaves y hover effects en tarjetas y enlaces.

En conjunto, este proyecto refleja nuestro aprendizaje, trabajo en equipo y pasión por la cocina. Esperamos que disfruten navegando por Delicias del Mundo tanto como nosotros disfrutamos creándolo.
