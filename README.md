# TrabajoEvaluativo

TrabajoEvaluativo: Recetas del Mundo (The Meal DB API)
Descripcion General
Este proyecto es un sitio web adaptativo (responsive) creado para una evaluación del curso de Desarrollo Web 2025. El objetivo principal fue integrar una API pública (The Meal DB) para mostrar datos de recetas y asegurar que el diseño sea adaptable a cualquier dispositivo (móviles, tabletas y escritorio) usando HTML, CSS y JavaScript.

Tecnologias Utilizadas
HTML5: Usado para la estructura principal y la semántica de las tres páginas: index, contacto y acerca de nosotros.

CSS3: Se encarga de todo el diseño. Incluye el uso de Flexbox y Media Queries para lograr la responsividad. También se usaron variables CSS para manejar colores y fuentes de manera uniforme.

JavaScript: Maneja la lógica. Es el responsable de conectar con la API, de cargar las recetas dinámicamente en el index y de las validaciones en el formulario de contacto.

API: The Meal DB (https://www.google.com/search?q=themealdb.com). Es la fuente de la información de las recetas.

Caracteristicas Implementadas
Integracion con API: El index carga recetas de la categoría 'Seafood' directamente desde The Meal DB.

Diseño Adaptativo: El sitio web se ajusta automáticamente a diferentes tamaños de pantalla, desde el celular hasta la PC, gracias a las Media Queries.

Estilos: Se realizaron ajustes en la opacidad del footer y en la tipografía para mejorar la presentación del sitio.

Formularios: La página de contacto incluye un formulario con validaciones básicas implementadas con JavaScript.

Como Usar
Clonar el Repositorio: Usa el comando git clone para descargar el codigo a tu computadora.

Abrir el Archivo: Abre el archivo index.html directamente en tu navegador web. El JavaScript cargará las recetas automáticamente.
